# Fleuropa – Base de données et application web

## Description
Conception et développement d’une base de données relationnelle et d’une application web dans le cadre d’un projet académique.

Le projet s’appuie sur le cas de l’entreprise Fleuropa, spécialisée dans la livraison de fleurs, et vise à gérer les commandes clients, la préparation en magasin et la livraison.

## Fonctionnalités
- Gestion des clients (inscrits ou non)
- Gestion des commandes et des livraisons
- Suivi des commandes en temps réel
- Gestion des magasins et répartition géographique
- Gestion des modes de livraison (coursier / Chronopost)
- Intégration d’un système de suivi des commandes

## Technologies
- PostgreSQL
- Python
- Flask
- SQL

## Contexte du projet
Projet réalisé en binôme dans le cadre de la Licence Informatique.

Participation à la conception de la base de données et au développement de l’application web Flask, à partir d’un cahier des charges simulant une entreprise de e-commerce.

## Résultat
Projet académique validé avec une note de 15/20.

## Compétences développées
- Modélisation de base de données relationnelle
- Conception de schémas SQL (tables, contraintes, relations)
- Développement d’une application web avec Flask
- Gestion des données et requêtes SQL
- Compréhension d’un système d’information complet
